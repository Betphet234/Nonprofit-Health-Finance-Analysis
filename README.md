# Nonprofit Health Organization Finance Dashboard — 2023

This project analyzes the financial performance of nonprofit health organizations using 2023 filing data. The dashboard focuses on revenue, expenses, surplus/loss, profit margin, executive compensation, state-level performance, and organizational outliers.

The project includes both an interactive HTML dashboard and an Excel interactive dashboard, allowing users to explore nonprofit health finance data through filters, charts, KPI cards, and downloadable CSV files.

## Project Overview

The analysis focuses on fiscal year 2023 because it is the most complete year in the dataset. The goal is to provide a clear, interactive view of nonprofit health organization finances across the United States.

Users can explore:

* Total revenue, expenses, surplus, and margin
* Top nonprofit health organizations by revenue
* State-level financial performance
* Organizations with the largest surpluses and losses
* Revenue concentration among top organizations
* Executive compensation patterns
* High and low margin outliers

## Main Insights

The 2023 nonprofit health finance dataset shows that revenue is highly concentrated among a small number of large organizations. State-level totals vary significantly, with some states showing strong aggregate margins while others show overall losses. Executive compensation data is available for only a limited portion of organizations, so compensation findings should be interpreted carefully.

## Purpose

This project was created to demonstrate data cleaning, financial analysis, dashboard development, and interactive reporting using real-world nonprofit health finance data.
